홈페이지

  • 현재는 홈페이지 제작을 받고 있지 않습니다.
  • 필요한 경우 메일로 별도 문의해 주세요.
아이콘
  • 아이콘은 개당 가격으로 책정됩니다.
  • 번들 작업이 필요한 경우 필요한 아이콘 종류, 사용처, 메인 컬러, 콘셉트를 정확히 알려 주세요.
  • 화려하거나 세밀한 묘사가 필요한 경우 추가금이 발생할 수 있습니다.
  • 제작 기간은 1주일~1개월이며, 수량에 따라 변동됩니다.
개인 커미션
기본 안내
  • 모든 저작권은 저에게 있으며, 활용 시 반드시 출처를 밝혀 주세요.
  • 선입금은 필수입니다.
  • 상업적 용도 사용은 불가합니다. 외주 작업은 메일로 문의해 주세요.
  • 단순 변심에 의한 환불 및 수정은 불가합니다.
  • 작업 기간을 초과할 경우 전액 환불해 드립니다.
  • 고어 / 성인물 / 퍼리 캐릭터는 작업하지 않습니다.
  • 작업물은 샘플 및 개인 포트폴리오에 사용될 수 있습니다.
  • 필요한 규격(크기)이 있다면 반드시 사전에 알려 주세요. (기본: 300px / iPhone X 사이즈)
작업 기간 / 신청 양식
  • 작업 기간: 기본 7일~1개월
  • 빠른 마감: +100% 추가금 (최소 2일 기준)
  • 신청 양식: 타입 / 캐릭터 정보(이미지 가능) / 인원 수(1인 또는 2인) / 원하는 자세
  • ↳ A타입 동글이 / 캐릭터 정보(이미지 가능) / 1인 / 원하는 자세
  • ↳ B타입 동글이 풀채색 / 캐릭터 정보(이미지 가능) / 2인 / 원하는 자세
  • 배경 없음: +0.3
타입 안내
  • 풀채색: 머리부터 발끝까지 채색합니다.
  • 포인트: 포인트만 컬러로 작업하고, 나머지는 흑백으로 작업합니다.
  • 낙서: 가볍게 흑백 처리되며, 선 정리가 깔끔하지 않을 수 있습니다.
  • 동글이: 동그랗게 크롭했을 때 귀엽게 보이도록 작업하며, 두상만 작업합니다.
가격표 (A 타입 / SD)
  • SD 작업물은 동글이 타입을 제외하고 모두 전신만 가능합니다.
  • 풀채색
  • ↳ 3.5
  • 포인트
  • ↳ 2.0
  • 낙서
  • ↳ 0.7
  • 동글이
  • ↳ 1.0
가격표 (B 타입 / LD)
  • 현재 LD 풀채색은 받지 않습니다.
  • 포인트
  • ↳ 전신 4.0 / 반신 2.5 / 두상 1.5
  • 낙서
  • ↳ 전신 3.5 / 반신 2.0 / 두상 1.0
  • 동글이
  • ↳ 풀채색 1.5 / 포인트 1.0
추가 안내
  • 복잡한 의상/액세서리의 경우 추가금이 발생할 수 있습니다.
  • 캐주얼 스타일 특성상 복잡한 디자인은 생략되거나 단순화될 수 있습니다.
  • 꼭 포함하고 싶은 포인트가 있다면 신청 시 명확하게 적어 주세요.
